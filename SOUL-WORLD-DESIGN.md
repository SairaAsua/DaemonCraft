## 🏰 La Casa-Castillo: "El Bodygraph de Ra"

### Principios Arquitectónicos

#### 1. Proporción Áurea (φ = 1.618...)

Todas las dimensiones principales siguen la proporción áurea:

| Elemento | Ratio | Ejemplo (bloques MC) |
|----------|-------|---------------------|
| Sala del Trono (Voz) | φ:1 | 34×21 |
| Biblioteca HD | φ:1 | 21×13 |
| Cuartos privados | 1:φ | 13×21 |
| Torre del Oráculo | φ²:1 | 55×21 |
| Jardín Central | Espiral áurea | Radio 13 → 21 → 34 |
| Pasillos | 1:1 (raíz cuadrada) | 8×8 |

#### 2. La Espiral Áurea como Plano

El jardín central se construye como una **espiral áurea de Fibonacci**:
```
  [8] [13]
[5]     [21]
  [3] [2] [1] [1]
```
Cada cuadrado es un sector de jardín con un bioma diferente:
- 1: Centro — Fuente de agua (el Umbral)
- 1: Cactus (el Desierto del Ser)
- 2: Bambú (el Canal de la Respuesta)
- 3: Flores del valle (el Jardín Emocional)
- 5: Árboles de azalea (el Bosque del Ego)
- 8: Micelio/luz (el Reino del Inconsciente)
- 13: Nieve/hielo (la Corona del Espíritu)
- 21: Pradera dorada (el Sacro)

#### 3. Los 9 Centros como Habitaciones

| Centro | Habitación | Función | Color/Elemento |
|--------|-----------|---------|----------------|
| Corona | Torre del Oráculo | Observación, meditación | Púrpura, cristal |
| Cabeza | Biblioteca HD | Conocimiento, estudios | Azul, libros |
| Ajna | Sala de las Puertas | Mapas, planificación | Verde, mapas |
| Garganta | Sala del Voz (Ra) | Reuniones, palabra | Turquesa, campanas |
| Ego | Sala del Coraje | Logros, trofeos | Amarillo, oro |
| Corazón | Cuarto del Amor | Descanso, intimidad | Rojo, camas |
| Sacro | Cocina/Taller | Creación, trabajo | Naranja, hornos |
| Bazo | Bodega/Almacén | Recursos, seguridad | Marrón, cofres |
| Raíz | Cripta/Memoria | Archivo, historia | Negro, obsidiana |

#### 4. Las 64 Puertas como Ventanas

La fachada principal tiene **64 ventanas** — una por cada puerta del I Ching:
- 8 filas × 8 columnas
- Cada ventana tiene un color/estilo según su hexagrama
- Las ventanas definidas del castillo brillan de noche
- Las ventanas abiertas cambian según el tránsito oracular del día

#### 5. Canales Activos como Pasillos

Los pasillos que conectan las habitaciones siguen los **canales electromagnéticos**:
- Pasillos anchos = canales compartidos entre múltiples almas
- Pasillos estrechos = canales individuales
- Puentes = puntos de encuentro entre centros definidos

## 📑 Optimización de Contexto para Gemma

### Configuración de los 5 Bots

Todos los landfolk usan Gemma via Ollama:

```yaml
model:
  default: gemma-andy:e4b-v1-q8_0
  provider: openai
  base_url: http://10.10.20.1:11434/v1
  api_key: dummy
```

### Ventana de Contexto por Bot

| Bot | Rol | Context tokens | Estrategia |
|-----|-----|----------------|------------|
| Stevie | Líder/Arquitecto | 8192 | Prompt corto, plan detallado |
| Moss | Guía/Feng shui | 8192 | Prompt poético, observación |
| Reed | Mediador | 4096 | Prompt minimalista, reflejo |
| Flint | Trabajador | 6144 | Prompt acción, herramientas |
| Ember | Cuidadora | 8192 | Prompt emocional, ondas |

### Técnicas de Optimización

1. **Prompts modulares**: Cada bot carga solo su SOUL relevante
2. **Memoria comprimida**: Los 5 comparten un story-state.json común
3. **Sincronización oracular**: Un solo heartbeat alimenta a los 5
4. **Chat compartido**: El canal de chat es único, cada bot filtra por relevancia
5. **Planificación distribuida**: Stevie asigna tareas, los demás ejecutan

### Heartbeat único para 5

```python
# En agent_loop.py, antes de cada turno:
oracle_state = load_oracle_state()  # Lee transits/eko_current.json

# Inyecta solo el mensaje relevante para cada tipo:
if agent.type == "Manifestor":
    prompt += oracle_state["initiation_msg"]
elif agent.type == "Projector":
    prompt += oracle_state["guidance_msg"]
elif agent.type == "Reflector":
    prompt += oracle_state["reflection_msg"]
elif agent.type == "Generator":
    prompt += oracle_state["response_msg"]
```
