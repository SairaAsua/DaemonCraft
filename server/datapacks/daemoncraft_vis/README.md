# DaemonCraft Visibility Datapack

Vanilla Minecraft datapack for the DaemonCraft server. No client mods required.

## Features

- **Coordinates HUD** — Shows live X/Y/Z coordinates in the action bar for all players
- **Colored Player Markers** — Each agent/player gets a unique team color + permanent glowing effect, so you can see everyone through walls

## Installation

Copy this folder into your world's `datapacks/` directory:

```bash
cp -r server/datapacks/daemoncraft_vis /path/to/world/datapacks/
```

Then reload in-game or via console:

```
/reload
```

## Minecraft Version

Tested on 1.20.1 (pack_format 15).
